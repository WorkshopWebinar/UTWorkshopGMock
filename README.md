# UTWorkshopGMock

# GTest/GMock installation

$ git clone https://github.com/google/googletest.git

$ cd googletest/

$ mkdir build

$ cd build/

$ cmake ..

$ make

$ sudo make install


# Run build and test result

$ mkdir build

$ cd build

$ cmake ..

$ make

$ ./GMockTestExamples
